# fitness-tracker


🚀 Fitness Tracker Project using Python

Built a simple fitness tracking system that analyzes weekly step counts, calculates calories burned, checks fitness goals, and visualizes the data using graphs.



🔹 Features:

• Weekly step tracking

• Calories burned calculation

• Goal achievement analysis

• Average step count insights

• Data visualization with charts



🛠️ Tech Stack:

🐍 Python

📊 NumPy

📋 Pandas

📈 Matplotlib



This project helped me strengthen my skills in Python programming, data analysis, and visualization.
